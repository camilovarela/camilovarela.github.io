¡Te reto! Construye tu propio blog

Camilo Varela, Jan 14, 2021, Personal Blog GitHub Pages Hugo Static Web Site

Desde hace un par de años tenía el objetivo de comenzar a escribir un blog tecnológico con el propósito de compartir mis ideas alrededor del tema que más me apasiona en la vida: El software y todas sus aristas. La escritura me parece que es un arte en sí misma, y el acto de escribir -así sea por simple afición-, nos permite repisar y afianzar nuestro propio conocimiento como también nos da el poder de compartirlo y transmitirlo.

Intenté lograr este objetivo en un par de ocasiones en el pasado sin mayor resultado que el de encontrarme frente a la pantalla, con la mente totalmente en blanco, con todas las herramientas configuradas y preparadas para iniciar, pero sin saber por dónde comenzar o sin siquiera saber sobre qué escribir. El objetivo parecía diluirse año tras año tras y parecía no tener una causa real más allá de la procrastinación natural en muchos de nosotros… solo hasta hoy.

Así pues, me armé de motivación, comencé a leer sobre herramientas y tecnologías para construir blogs y me encontré con varias alternativas. Un blog es en sí mismo un sitio web y para construir uno podemos irnos por una de las siguientes dos variantes: sitios web dinámicos o sitios web estáticos. Es sobre ésta última alternativa con la que vamos a trabajar.

¿El reto? En las próximas semanas voy a estar publicando, semanalmente, un artículo que te ayudará en el proceso de construcción de tu blog personal utilizando Hugo, una herramienta SSG (Static Site Generator) para construir sitios estáticos a partir de plantillas. Al finalizar contarás con tu sitio web publicado en internet -similar a este que estás viendo-, tu dominio personal (opcional), y una configuración mínima de automatización como introducción al mundo DevOps.

En los 3 artículos propuestos, abordaremos los siguientes temas:

Artículo 1: Hugo + GitHub Pages

En esta guía podrás seguir el paso a paso para crear la primera versión de tu blog, con tareas que van desde la instalación y configuración de las herramientas, jugar con los temas y darle a tu sitio un toque personal, hasta finalmente lograr su publicación en el servicio de hosting de Github llamado GitHub Pages.

Artículo 2: Google domains

Haremos la configuración de tu propio dominio midominio.com, administrado por GoogleDomains, y garantizaremos la integración con el dominio de Github Pages.

Artículo 3: DevOps, ¡al ataque!

Breve introducción a DevOps y cómo automatizar la publicación de nuestro sitio web con GitHub Actions y Azure Devops.

Mientras llega la próxima semana te comparto algunos recursos para que vayas leyendo.
